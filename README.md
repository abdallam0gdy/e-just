# EduAttend - نظام تسجيل الحضور الإلكتروني 🎓

نظام متكامل لتسجيل حضور الطلاب للجامعة المصرية اليابانية للعلوم والتكنولوجيا (E-JUST)، باستخدام التحقق عبر الصور (Selfie + ID Card) والمصادقة الآمنة.

## 🚀 المميزات الأساسية

- **نظام أدوار متعدد:** Role-Based Access Control مع 3 أدوار (أدمن، دكتور، طالب)
- **تسجيل حضور بالصور:** التقاط سيلفي + صورة الكارنيه من الكاميرا مباشرة
- **إدارة المحاضرات:** إنشاء/تعديل/حذف محاضرات مع دعم التكرار الأسبوعي
- **تخزين سحابي:** رفع الصور على ImgBB لتوفير مساحة قواعد البيانات
- **لوحة تحكم الأدمن:** متابعة وتصفية سجلات الحضور بالاسم والتاريخ والمحاضرة
- **حماية من التسجيل المزدوج:** منع الطالب من تسجيل حضور مرتين لنفس المحاضرة
- **تصميم تفاعلي RTL:** واجهات احترافية تدعم العربية ومتوافقة مع جميع الشاشات
- **توقيت مصر:** كل المقارنات الزمنية تتم بتوقيت القاهرة (Africa/Cairo)

## 🗄️ قاعدة البيانات

### الجداول

- **`profiles`**: بيانات المستخدمين (الاسم، البريد، الدور)
- **`attendance_logs`**: سجلات الحضور (الطالب، المحاضرة، صورتين، الوقت)
- **`lectures`**: المحاضرات (العنوان، التاريخ، الوقت، التكرار)

### الأدوار وصلاحياتها

| الدور | تسجيل حضور | عرض السجلات | إدارة المحاضرات | حذف السجلات |
|-------|------------|-------------|-----------------|-------------|
| 🎓 طالب | ✅ | سجلاته فقط | ❌ | ❌ |
| 👨‍🏫 دكتور | ❌ | الكل | ✅ | ❌ |
| 🛡️ أدمن | ❌ | الكل | ✅ | ✅ |

### Database Functions (Supabase)

| الدالة | الوظيفة |
|--------|---------|
| `get_today_lectures()` | جلب محاضرات اليوم (بتوقيت القاهرة) |
| `get_active_lectures_now()` | جلب المحاضرات النشطة الآن |
| `get_server_time()` | جلب وقت السيرفر (TIMESTAMPTZ) |
| `generate_recurring_lectures()` | توليد المحاضرات المتكررة أسبوعياً |

## 📦 التشغيل المحلي

1. تثبيت الاعتمادات:
   ```bash
   npm install
   ```

2. إنشاء ملف `.env` (انسخ `.env.example`):
   ```env
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   VITE_IMGBB_API_KEY=your_imgbb_api_key
   ```

3. تطبيق مخطط قاعدة البيانات في Supabase SQL Editor:
   - شغّل `supabase_schema.sql` أولاً
   - ثم شغّل `supabase_lectures_migration.sql`

4. تشغيل خادم التطوير:
   ```bash
   npm run dev
   ```
